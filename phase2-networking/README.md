# Phase 2: Networking

This phase builds the secure network infrastructure for the three-tier application architecture.

## Topics Covered
- **Virtual Networks (VNets)** – Isolated network environments for Azure resources
- **Subnets** – Network segmentation for web, app, and data tiers
- **Network Security Groups (NSGs)** – Traffic filtering rules for inbound/outbound traffic
- **DNS Configuration** – Name resolution for internal and external resources
- **Route Tables** – Custom routing for traffic control

## Skills Demonstrated
- Designing hub-spoke or multi-tier network architectures
- Implementing NSG security rules
- Configuring Azure DNS and private DNS zones
