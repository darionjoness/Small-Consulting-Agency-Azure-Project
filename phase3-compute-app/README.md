# Phase 3: Compute & Application

This phase deploys the application workloads including web and serverless compute services.

## Topics Covered
- **Azure App Service** – Hosting the web tier application
- **Azure Function App** – Serverless compute for the app tier
- **Storage Account** – Data storage with secure access
- **Private Endpoints** – Secure, private connectivity to Azure services

## Skills Demonstrated
- Deploying and configuring Azure App Service
- Building serverless solutions with Azure Functions
- Implementing private endpoints for enhanced security
- Integrating storage with applications
