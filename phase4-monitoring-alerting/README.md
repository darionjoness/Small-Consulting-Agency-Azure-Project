# Phase 4: Monitoring & Alerting

This phase implements observability and proactive alerting for the Azure environment.

## Topics Covered
- **Log Analytics Workspace** – Centralized log collection and analysis
- **Azure Monitor Alerts** – Proactive notifications for critical events
- **Dashboards** – Visual representation of key metrics and KPIs
- **KQL (Kusto Query Language)** – Querying logs for insights and troubleshooting

## Skills Demonstrated
- Configuring diagnostic settings for Azure resources
- Writing KQL queries for log analysis
- Creating action groups and alert rules
- Building custom Azure dashboards
