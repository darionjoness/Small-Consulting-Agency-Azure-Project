# Phase 1: Governance & Identity

This phase establishes the foundational governance and identity controls for the Contoso Consulting Azure environment.

## Topics Covered
- **Resource Groups** – Logical organization of Azure resources
- **Tagging Strategy** – Consistent metadata for cost tracking and management
- **RBAC (Role-Based Access Control)** – Assigning permissions using least-privilege principles
- **Azure Policies** – Enforcing organizational standards and compliance

## Skills Demonstrated
- Implementing Azure governance best practices
- Configuring custom RBAC roles
- Applying and auditing Azure Policy assignments
